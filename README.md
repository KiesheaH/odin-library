# 📚 Library App

A small JavaScript application that allows users to add and remove books in a personal library.

This project was built as part of the JavaScript course from The Odin Project.

# 🎯 Project Objectives

The goal of this project was to practice:

- Object constructors
- Manage application state using arrays
- DOM manipulation
- Event handling
- Separating data logic from UI rendering
